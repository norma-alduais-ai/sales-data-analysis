## Project Overview
This project focuses on analyzing sales data to extract meaningful business insights.
The analysis includes data cleaning, descriptive statistics, and basic exploratory data analysis (EDA).

## Dataset
- **File:** `sales_data.csv`
- **Rows:** 1000
- **Columns:** 14
- Contains information about sales amount, quantity sold, regions, products, discounts, and sales channels.

A second dataset (`sales_data_dirty.csv`) was created to practice data cleaning techniques.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning
- Identified missing values in multiple columns
- Practiced:
  - Dropping rows with missing values
  - Evaluating when to drop vs fill missing data

## Descriptive Statistics
Calculated:
- Mean
- Median
- Standard Deviation
- Minimum & Maximum values

For:
- Sales Amount
- Quantity Sold
- Unit Cost
- Unit Price
- Discount

## Key Insights
- Average sales amount is around **5000**
- Quantity sold varies significantly across transactions
- Discounts range from **0% to 30%**
- Unit price is consistently higher than unit cost, indicating profit margins
